Для выполнения задания у вас должен быть установлен для Windows - GitBash.
Создан аккаунт в GitHub
Создан репозиторий в GitHub
Созданный репозиторий склонирован на локальный компьютер в отдельную папку.

Все шаги сценария выполняйте в терминале GitBush, Terminal, в папке под гитом.


Как отправить ДЗ на проверку.
 1. Создайте текстоовый файл как в первом ДЗ по Terminal.
 2. Сценарий перенесите в этот файл.
 3. На против каждого действия - напишите команду в GitBash

Файл со сценарием и ссылку на свой гит хаб отправляйте менторам на проверку.

 4. Создать файл “new.json” - touch new.json
 5. Добавить файл под гит. - git add .
 6. Закоммитить файл. - git commit -m "add new.json files"
 7. Отправить файл на внешний GitHub репозиторий  -   git push
 8. Создать файлы new2.json, new2.xml, new2.txt   - touch new2.json new2.xml new2.txt
 9. Добавить файлы под гит -  git add .
 10. Закоммитить файлы - git commit -m "new2.json,new2.xml,new2.txt"
 11. Отправить файлы на внешний GitHub репозиторий - git push

 12. Отредактировать содержание файла “new2.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). - vim new2.txt

 "FirsName": "Kate",
 "LastName": "Kastsian",
 "Ages": 29,
 "PetsNumber": 0,
 "ExpectedSalary": 1700$
}
 14. Добавить и закоммитить “new2.txt” одной строчкой. - git add new2.txt && git commit -m "by 1 string"
 15. Добавить и закоммитить “new2.json” одной строчкой. - git add new2.json && git commit -m "by 1 string"
 16. Отправить изменения на внешний репозиторий - git push
